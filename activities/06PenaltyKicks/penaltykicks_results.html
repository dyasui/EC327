Dataset Structure

I collected the results from the in-class penalty kicks activity as a csv file where each row represents an different attempted shot.

The columns round, match, and shot contain the id representing which round in the bracket, which match of that specific round, and whether that shot was the first, second, third, fourth, or fifth shot of that match, respectively.

To make these data easier to visualize and calculate statistics, you can import them to an object in R using the read_csv function from the tidyverse package.

# load tidyverse packages
library(tidyverse)

# setwd("activities/06PenaltyKicks")

data <- read_csv("EC327F24-activity6_results.csv") %>%
  filter(!is.na(s_k), !is.na(s_g)) # filter out missing data

The important variables are the choices of the kicker and goalie for each kick attempt as well as whether a goal was scored or not:

  • s_k represents the direction the kicker chose to kick
  • s_g represents the direction the goalie chose to attempt to block
  • goal is an indicator which is TRUE if a goal was scored, and FALSE if a kick was blocked or missed.
N <- nrow(data)

Across all rounds and matches played in class this quarter, we had 324 different shots attempted, which are each a row in this dataset.

Group Results by Srategy Profile

I might want to compare how often we see a kicker choosing \(Right\) at the same time a goalie blocked \(Right\), vs a kicker chosing \(Right\) when a goalie blocked \(Left\), etc, for all 9 different possible pure strategy combinations.

See the code 
# summarize by type of strategy profile
profiles_tbl <- data %>%
  group_by(s_k, s_g) %>% # group by each strategy combo to summarise each group
  summarise(
    n = n(), # get counts of each profile
    p = n / N, # calculate proportions of total observations
    num_goals = sum(goal), # count numbers of goals scored
    prob_goal = num_goals / N # probability of goal scored
  ) %>%
  ungroup() # get rid of groups structure to make later manipulations easier

Testing strategy profiles against theory

If you wanted to compare the frequency of each type of choice matchup in this table with what you would expect in an MSNE, you could use the chisq.test function to test the goodness of fit of the n column in our table to some theoretical distribution of p which we would expect for each pure strategy combination.

For example, suppose that a mixed strategy Nash equilibrium of this game would result in each profile being observed with equal probability of 1/9.

See the code 
# use observed num of each choice combo from profiles table
n_observed <- profiles_tbl %>% select(n)
# theoretical predictions of each profile being observed
# make sure they're in the same order as in the profiles table
p_predicted <- c(1/9, 1/9, 1/9, 1/9, 1/9, 1/9, 1/9, 1/9, 1/9)
# test whether n is drawn from p_predicted distribution
chisq.test(
  x = n_observed,
  p = p_predicted
)
